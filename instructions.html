Timeline Generator Guide

1. Running the Program

To run the program, navigate to the directory containing the timeline_generator.py script. Then execute the following command:

python timeline_generator.py filename.csv

Replace filename.csv with the name of your input CSV file.

2. Building Your Configuration File

The configuration file allows you to define and customize events based on the operations found in your CSV.

An example of a configuration snippet:


        {
          "events": {
            "OPERATION_CODE": {
              "name": "Event Name",
              "description": "This event occurred due to {reason} at location {location}.",
              "source": "Event Source"
            }
          }
        }

In the above example, OPERATION_CODE corresponds to an operation value from your CSV. The placeholders {reason} and {location} are dynamic and will be replaced by the corresponding columns from the CSV.

When creating your own configuration, ensure you:

  • Match the operation codes from your CSV to those in the configuration.
  • Use placeholders (in curly braces) to match CSV column names for dynamic insertion.

Any unmatched operation codes or placeholders will result in default "Unknown" values in the generated timeline.
